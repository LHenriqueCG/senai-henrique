Como importar para o Google Forms

Passo a passo para criar o formulário de avaliação no Google Forms com todas as 40 questões.

📋 Passos para criar o Google Forms

Passo 1Acesse forms.google.com e clique em "+" para criar um novo formulário.
Passo 2Dê o título: "Avaliação — Programador Front-End | Módulo Básico".
Passo 3Adicione uma seção por aula (botão de seção). Nomeie cada seção com o título da aula.
Passo 4Para cada pergunta, escolha o tipo "Múltipla escolha" e cole a pergunta + alternativas desta página.
Passo 5Ative "Gabarito" (botão roxo "Questionário" nas configurações) e marque a alternativa correta em cada questão.
Passo 6Nas configurações, ative "Reúna endereços de e-mail" para identificar os 31 alunos.
